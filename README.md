# Dark Room - Photo Editing Program

## Overview
Photo editing application utilizing compute shaders for GPU-accelerated image processing.

## Current Features
- GPU-accelerated image processing
- histogram that updates dynamically as exposure and RGB values are changed
- HSL slider
- adjustable zoom
- standard image editing features: exposure, white, black, hihglights, contrast, and shadows
- Can edit 4k images without stutter

## Tech Stack
- C++
- C
- JavaScript
- OpenGL / graphics pipeline tools
- IMGUI for user interface
